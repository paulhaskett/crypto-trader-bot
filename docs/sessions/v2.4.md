# Session: v2.4 - Trailing Stop Implementation

## Summary

Implemented a dynamic trailing stop system that:
- Never sells at a loss (break-even floor)
- Locks in profit as price increases (trailing from peak)
- Uses per-regime trailing percentages (12% uptrend, 5% neutral, 3% downtrend)
- Fetches actual fee rates from Coinbase API weekly

## Key Implementation Details

### 1. Fee Fetching & Storage

- **API**: `GET /api/v3/brokerage/transaction_summary`
- **Storage**: `user_settings` table with keys `coinbase_maker_fee`, `coinbase_taker_fee`, `fee_rates_updated_at`
- **Frequency**: Startup + every 7 days
- **Fallback**: 0.6% maker, 1.2% taker (intro tier) if API fails

### 2. Per-Regime Trailing Stop

| Regime | Trailing % | Behavior |
|--------|-----------|----------|
| `uptrend` | 12% | Let winners run |
| `neutral` | 5% | Moderate protection |
| `downtrend` | 3% | Tighter in falling market |

### 3. Trailing Stop Logic

```python
# 1. Calculate break-even (covers fees)
break_even = entry_price × (1 + maker_fee + taker_fee)

# 2. Initial floor (10% below entry)
initial_stop_floor = entry_price × 0.90

# 3. Get regime and trailing percentage
regime = position.get('regime', 'neutral')
trailing_pct = TRAILING_STOP_REGIME_MAP[regime]

# 4. Get peak (initialize at entry)
peak_price = position.get('peak_price', entry_price)

# 5. Update peak ONLY if in profit (above break-even)
if current_price >= break_even and current_price > peak_price:
    peak_price = current_price

# 6. Calculate trailing stop
if current_price >= break_even:
    trailing_stop = peak_price × (1 - trailing_pct)
else:
    trailing_stop = entry_price  # No trailing below break-even

# 7. Final stop = max(trailing_stop, break_even, initial_stop_floor)
# This ensures we NEVER sell at a loss
final_stop = max(trailing_stop, break_even, initial_stop_floor)
```

### 4. Peak Activation

Peak only starts moving when:
```
current_price >= break_even
```

This ensures:
- Trailing stop doesn't activate until position covers fees
- First profit is "locked in" before trailing begins
