## New Features

### 1. WebSocket Real-Time Prices
- Implemented Coinbase WebSocket client for real-time price feeds
- High-liquidity pairs (BTC, ETH) now use WebSocket
- Falls back to REST if WebSocket unavailable
- Per slippage-modeling.md requirements

### 2. Scale-In Entry Price Fix
- Previously: Scale-in would update entry_price to weighted average
- Now: Keeps original entry_price, only updates weighted_entry_price for P&L
- Preserves original entry date for tax purposes

### 3. Deviation Threshold
- Changed from 3% to 2% per slippage-modeling.md
