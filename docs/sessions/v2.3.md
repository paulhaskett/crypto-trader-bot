## Problem 1: SQL NOT NULL Constraint Failed

### Issue
```
NOT NULL constraint failed: open_positions.side
```

### Root Cause
Two places were updating open positions without including the `side` field:
1. `trading_engine.py:162` - Wallet sync update
2. `risk_manager.py:797` - Portfolio risk sync

### Fix

**trading_engine.py** (line 162):
```python
# Before
db_manager.save_open_position({
    'product_id': product_id,
    'size': wallet_balance,
    'entry_price': entry_price,
    'weighted_entry_price': entry_price
})

# After - preserve side from existing
db_manager.save_open_position({
    'product_id': product_id,
    'side': existing.get('side', 'buy'),
    'size': wallet_balance,
    'entry_price': entry_price,
    'weighted_entry_price': entry_price
})
```

**risk_manager.py** (line 797):
```python
# Load existing position to get current side
existing_positions = db_manager.load_open_positions(trade_type='live')
```

## Problem 2: Agreement Matrix N/A

### Issue
Models page showed N/A for all individual model signals (rf_signal, gb_signal, etc.)

### Root Cause
The signal cache file (`signal_cache.json`) only stored:
- `action`, `confidence`, `regime`, `reason`, `timestamp`

Individual model predictions (`rf_prediction`, `gb_prediction`, etc.) were generated but not saved to cache.

### Fix

Extended signal cache writing in `api_worker.py`:

```python
# write_signal_cache_to_file() - now includes individual model predictions
cache_data[product_id] = {
```

## Agreement Matrix - Dynamic

The ensemble agreement is calculated dynamically at prediction time:

```python
# Contextual agreement - weight based on actual votes
majority = vote_counts.most_common(1)[0][0]

for v in vote_list:
    if v == majority:
        vote_values.append(1.0)  # Exact match
    elif abs(v - majority) == 1:
        vote_values.append(0.75)  # HOLD is close to BUY/SELL
    else:
        vote_values.append(0.0)

agreement = sum(vote_values) / n_models
```

## Ensemble Weights - Dynamic

Weights are calculated from training F1 scores when `ENSEMBLE_WEIGHT_MODE = 'performance'`:

```python
# config/settings.py
ENSEMBLE_WEIGHT_MODE: str = 'performance'  # or 'manual'

# Manual weights (if mode = 'manual')
RF_WEIGHT: float = 0.35
```
