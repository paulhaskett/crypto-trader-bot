## Problem 1: Trailing Stop Wrong P&L

### Issue
ETH position was closed at a "profit" of £2.46 (9.1%) when it was actually a loss of £0.39.

### Root Cause
Used `entry_price` from open_positions table (last scale-in price: £1,577.53) instead of true FIFO cost basis (£1,743.38).

### Fix
```python
# In monitor_positions(), get TRUE cost basis from Coinbase fills
if not self.paper_trading:
    cost_basis_size, fifo_entry_price = get_fifo_cost_basis(product_id)
    if fifo_entry_price > 0 and cost_basis_size > 0:
        entry_price = fifo_entry_price
        logger.info(f"Using FIFO cost basis for {product_id}: £{fifo_entry_price:.2f}")
```

## Problem 2: Binance 34% Outlier

### Issue
Binance prices showed 34% deviation - flagged as outlier.

### Root Cause
Binance doesn't have GBP trading pairs. Symbol `BTCGBP` doesn't exist - API returned stale cache data.

### Solution
1. Added `COINBASE_TO_BINANCE_USDT` mapping
2. Use USDT pair (e.g., `BTCUSDT`) + convert via implied exchange rate
3. Implied rate = USDT price / Coinbase GBP price (95% weight) + fiat rate (5% weight)
