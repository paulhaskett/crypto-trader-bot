## Architecture

### Multi-Source Price Flow
1. Trading engine requests current prices
2. MultiSourcePricer fetches from Coinbase, Kraken, CryptoCompare
3. Consensus algorithm calculates weighted average
4. Outliers (>3% deviation) are excluded
5. Confidence score calculated based on source agreement

### Key Features
- **Weighted Consensus**: Coinbase (45%), Kraken (30%), CryptoCompare (25%)
- **Outlier Detection**: Excludes prices >3% from median
- **Graceful Degradation**: Works with any combination of sources
- **60% Confidence Threshold**: Minimum confidence for trading signals

## Decisions

- [Consensus averaging] → Better than "best price" for reducing manipulation risk
- [Chainlink as verification only] → Too complex for direct integration, use CryptoCompare as alternative
- [Minimum 2 sources] → Ensures consensus has enough data points
- [Skip CoinGecko without API key] → Prevents rate limit issues, falls back to Kraken
