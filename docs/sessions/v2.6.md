## 2. Signal Sync Fix - Dashboard Now Shows Trading Signals

### Problem

- **Trading Engine**: Called `ai_model.get_signal()` directly each cycle (fresh signals)
- **Dashboard**: Read from `signal_cache.json` file (stale, only updated manually)
- Dashboard showed different signals than trading engine used

### Solution

Added `_sync_signals_to_cache()` method in trading_engine.py that writes fresh signals to cache file after each trading cycle:

```python
def _sync_signals_to_cache(self):
    """Write all current signals to cache file for dashboard synchronization."""
    try:
        signals_for_cache = {}
        for product_id in settings.PRODUCT_IDS:
            signal = ai_model.get_signal(product_id)
            signals_for_cache[product_id] = signal
        write_signal_cache(signals_for_cache)
    except Exception as e:
        logger.error(f"Error syncing signals to cache: {e}")
```

Called after each `run_trading_cycle()` completes.

## 3. Trailing Stop Sell Fix

### Problem

Trailing stop was updating position records but NOT executing actual sell orders on Coinbase.

### Root Cause

`_close_position()` only updated in-memory/D database records - it never called `execute_live_trade()`.

### Solution

Added actual sell execution in trailing stop logic:

```python
# Before (line 646):
self._close_position(position_id, pnl, exit_reason, current_price)

# After (lines 640-655):
if should_close:
    sell_size = position.get('remaining_size', position.get('size', 0))
    
    # Actually execute the sell order on Coinbase
    if not self.paper_trading and sell_size > 0:
        try:
            order_result = self.execute_live_trade(product_id, 'sell', sell_size)
            logger.info(f"[TRAILING STOP] Sell order result: {order_result}")
        except Exception as e:
            logger.error(f"[TRAILING STOP] Failed to execute sell: {e}")
    
    # Update position record in DB
    self._close_position(position_id, pnl, exit_reason, current_price)
```

### Behavior Now

| Trigger | Action |
|---------|--------|
| Trailing stop hit | Log "[TRAILING STOP] SELL TRIGGERED" |
| | Execute actual sell order on Coinbase |
| | Update position record in DB |
