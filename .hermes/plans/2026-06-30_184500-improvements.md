# Crypto Trader Bot - Improvements Plan

> **Status:** Ready for execution — modular chunks, pick up any chunk independently.

**Goal:** Reduce resource usage (RAM/CPU) on the Pi, fix wasted API calls, clean up codebase, improve reliability.

**Architecture:** Single-container Flask app + Gunicorn + SQLite, with a subprocess trading loop. Multi-source price aggregation (Coinbase, Kraken, Binance, CoinGecko, CryptoCompare) with consensus averaging. WebSocket only covers BTC-USD and ETH-USD.

**Pi Constraints:** 4 cores, ~8GB RAM total. Container limited to 1.5GB, currently using 1.09GB (72%). Background trading cycle runs every 4 hours. SD card I/O limited — avoid excessive logging.

---

## Chunk 1: Stop CryptoCompare API Waste (5 min)

**Objective:** CryptoCompare API returns 401 every cycle — silent failure that wastes CPU and bandwidth. Either fix credentials or disable.

**Files:**
- Modify: `/projects/crypto-trader-bot/src/multi_source_pricer.py`
- Modify: `/projects/crypto-trader-bot/config/settings.py`

**Steps:**

1. Check if `CRYPTOCOMPARE_API_KEY` is set in `/projects/crypto-trader-bot/config/api_keys.env`
2. If not set, add `CRYPTOCOMPARE_ENABLED = False` to settings.py
3. In `multi_source_pricer.py`, guard cryptocompare init: `if settings.CRYPTOCOMPARE_ENABLED: self.cryptocompare = get_cryptocompare_api()`
4. Restart container, check logs — no more 401 errors

**Verification:** `docker logs crypto-trader-bot 2>&1 | grep -i "crypto\|401\|error" | tail -5`

## Chunk 2: Reduce Memory Pressure (15 min)

**Objective:** Cut RAM usage from 1.09GB to ~700MB.

**Files:**
- Modify: `/projects/crypto-trader-bot/docker-compose.yml`
- Modify: `/projects/crypto-trader-bot/config/settings.py`

**Steps:**

1. `docker-compose.yml` — add `cpus: 2` under deploy.resources.limits
2. `settings.py` — change `LOG_LEVEL` from `DEBUG` to `INFO`
3. Reduce gunicorn workers from 3 to 2 in environment or startup.py
4. Rebuild: `docker compose build && docker compose up -d`
5. Verify: `docker stats crypto-trader-bot` — should show ~700-800MB
